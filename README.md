# Cartago Football Agency — Fullstack

This zip contains:
- Angular frontend
- .NET 8 Web API backend
- SQL Server schema
- JWT auth
- Public site without login
- Admin area with login
- View Profile modal window
- Players search + pagination
- Team photos included
- Club requests and player applications opening in dedicated admin detail pages
